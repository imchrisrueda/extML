En la carpeta 'k_brazos' se adjunta todo el código necesario para cumplir los requisitos definidos por el profesor sobre el problema del Bandido de k-brazos. 
En la carpeta 'src' se adjuntan todos los archivos python necesarios para la ejecución del código.
    * 'algorithms' contiene los algoritmos utilizados
    * 'arms' contiene los diferentes tipos de brazos, con diferentes distribuciones de probabilidad (normal, binomial, Bernoulli)
    * 'plotting' contiene los gráficos a mostrar relacionados con el algoritmo epsilon greedy
